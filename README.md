# bang bang

Work In Progress...

a small js / canvas game to attempt to create 2D maps which can be carved as in worms.

## Keys

    player 1:

        rotate CCW: left
        rotate CW:  right
        fire:       space

    player 2:

        rotate CCW: A
        rotate CW:  D
        fire:       S
